# Support

If you have questions or feedback about zb,
join [zb's Zulip chat](https://zb-build.zulipchat.com/).
The [\#support channel](https://zb-build.zulipchat.com/#narrow/channel/601347-support)
is specifically intended for helping zb users.
Please bear in mind that zb is still a small volunteer-driven project,
and so the availability of contributors is limited.

Those who participate in zb community spaces must follow the
[zb Code of Conduct](http://github.com/256lights/zb/blob/main/CODE_OF_CONDUCT.md).
